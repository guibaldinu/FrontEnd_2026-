# Atividade 10 — Jogo da Velha (React)

## Como abrir
Abra o `index.html` no navegador. Os três arquivos precisam estar na mesma pasta.

## Arquivos
- **index.html** — Página principal, carrega o React e monta o `<Game />`
- **game.js** — Componentes `Square`, `Board` e `Game` com toda a lógica do jogo
- **style.css** — Estilos do tabuleiro, células e histórico

## Funcionalidades
- Tabuleiro 3×3 com controle de vez entre X e O
- Detecta vencedor e destaca a linha vencedora em verde
- Histórico completo de jogadas — clique em qualquer jogada para voltar
- Usa `useState`, `createRoot` e `StrictMode` do React via `esm.sh`
