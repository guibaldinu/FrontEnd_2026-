# Atividade 8 — Laboratório de Lógica (Infinity Bank)

Sete exercícios de lógica e manipulação de DOM. Abra o `index.html` de cada pasta.

| Pasta | O que faz |
|---|---|
| Ex01 - Validador de CPF | Algoritmo dos dois dígitos verificadores, feedback verde/vermelho |
| Ex02 - Conversor de Temperatura | Celsius ↔ Fahrenheit em tempo real com `oninput` |
| Ex03 - Media Escolar | Média de 3 notas → Aprovado / Exame / Reprovado com cores |
| Ex04 - Taxas Infinity Bank | Taxa de bandeira com `switch`, juros simples 0,35%/parcela, taxa mensal R$12,50 |
| Ex05 - Lista Convidados VIP | Adicionar, concluir (riscar), editar com `prompt()`, excluir |
| Ex06 - Credito para Festas | Custo base + taxa 10% + desconto 5% para mais de 100 pessoas |
| Ex07 - Validador Cartao Luhn | Algoritmo de Luhn + identificação de bandeira e setor BIN/IIN |
